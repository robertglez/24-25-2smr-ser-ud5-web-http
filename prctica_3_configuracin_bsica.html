Práctica 3: Configuración básica

06.03: Configuració bàsica d'un node vitual

Objectius

Els objectius de l'activitat són:

  • Conèixer els paràmetres bàsics de configuració d'Apache2.
  • Distingir entre els diferents nivells de configuració (Global, per defecte, node virtual) que té Apache2.
  • Conèixer els diferents tipus de registres del servidor web Apache2.
  • Entendre el concepte de node virtual (virtual host).
  • Crear un node virtual

Supòsit

Per a realitzar la següent activitat ho farem des del supòsit següent: el nostre equip servirà de servidor d'allotjament de diversos llocs web. Per la qual cosa estructurarem la carpeta que conté els llocs web /var/www de forma que per a cada lloc web disposarem d'una carpeta que contindrà tant les pàgines html com els registres del servidor (logs). Esta carpeta tindrà el nom del domini a publicar. La imatge següent representa l'estructura de carpetes allotjant dos llocs web.

/var/www
├── html
│   └── index.html
├── secure.xarxa00.lan
│   ├── log
│   └── public_html
└── www.xarxa00.lan
    ├── log
    └── public_html


Per a cada lloc web es crearà un usuari en el sistema perquè puga administrar el contingut del lloc web. L'usuari tindrà el mateix nom que el domini (user_www) i sols tindrà accés via FTP a la carpeta del domini (www.xarxa00.lan).

Activitat (part 1): creació del lloc web virtual

  • El nou lloc web virtual respondrà al nom de domini www.xarxa##.lan.
  • Seguint l'esquema anterior crea les carpetes necessàries.
  • Crea un nou lloc virtual a partir del ja existent i ajusta després la configuració.
  • Edita també el servidor DNS si es necessari.
  • Penja un document HTML en el lloc web.
  • Comprova el funcionament.
  • Fent ús de la directiva ErrorDocument mostra un error personalitzat quan el codi d'estat siga 404 Not Found.
  • Documenta els canvis realitzats estructurant correctament el contingut.

Activitat (part 2): usuaris FTP

  • Crea un usuari específic per al lloc web amb els nom user_www.
  • Habilita-los accés mitjançant FTP a la carpeta del lloc web, perquè puguen gestionar el arxius del lloc web mitjançant FTP.
  • Documenta els canvis realitzant.

Recursos

En la Ubuntu Server Guide en l'apartat Opciones básicas disposes de la informació necessària per a fer la instal·lació i la configuració.
