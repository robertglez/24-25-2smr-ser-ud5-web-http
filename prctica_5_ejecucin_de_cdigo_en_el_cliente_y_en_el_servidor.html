Práctica 5: Ejecución de código en el cliente y en el servidor

06.05: Execució de codi. Client i Servidor

Objectiu

Activitat

Crea la següent pàgina web i guarda-la amb el nom: default.php

<html>
<body>
<h1>Data</h1>
<?php
  echo "<h2>Servidor (php)</h2>";
  echo date("r") ;
?>
<script type="text/javascript">
  var d = new Date();
  document.write("<h2>Client (javascript)</h2>"+ d.toString());
</script>
</body>
</html>

Accedeix a la pàgina mitjançant la URL: http://www.00.aula.local/default.php

Què ha passat? Per què?

Instal·la l'intèrpret PHP i el mòdul de PHP per a Apache2:

sudo apt install php libapache2-mod-php

Torna a accedir a la pàgina http://www.00.aula.local/default.php. Què ha canviat? Comprova la diferència amb l'execució anterior.
